Flexiv RDK Manual¶

Flexiv RDK (Robot Development Kit) is a powerful toolkit as an add-on to the Flexiv software platform. It enables the users to create complex applications with APIs that provide low-level real-time access to the robot.

All functionalities of RDK are packed into the Flexiv RDK library hosted on GitHub as flexiv_rdk repository at https://github.com/flexivrobotics/flexiv_rdk/.

The API documentation for Flexiv RDK library can be found at https://flexivrobotics.github.io/flexiv_rdk/.

In addition, a detailed documentation on available robot primitives can be found at https://flexivrobotics.github.io/primitives/.

Important

Before using the robot with Flexiv RDK, you must carefully read through all documents shipped with the robot as well as this RDK manual, and strictly follow all safety instructions detailed in these documents.

Key features¶

Feature

Summary

RT operation modes

The robot can execute low-level commands like joint torques, joint positions, Cartesian pose, etc. at 1kHz.

NRT operation modes

The robot can execute high-level commands like plans, primitives, digital IO, etc. as well as NRT motion commands.

RT states feedback

The robot sends back states data at 1kHz no matter what operation mode is being used.

Timeliness monitor

When in a RT operation mode, the robot checks if the user command arrives in time at 1kHz.

Integrated RT scheduler

Users can use this high-priority scheduler to run their own computation tasks in real time at 1kHz.

Integrated dynamics engine

Users can run-time access important robot dynamics data like mass matrix, gravity force, Coriolis force, etc.

Integrated visualization

Lightweight and easy-to-use visualizer based on Meshcat for users to see the robot(s) and scene as they run.

Portable C++ library

Self-contained static library that can be easily linked into a CMake project.

Portable Python library

Self-contained dynamic library that can be easily imported into a Python script.

Example pool

Plenty of informative examples covering almost every aspect of Flexiv RDK to help users quickly get started.

Robot description

Kinematics-only URDF and 3D mesh files of the robot for visualizers and simulators.
RT: real-time, NRT: non-real-time
